========================================
   EASIEST SOLUTION: USE ANDROID STUDIO
========================================

Since you have Android Studio installed, use it directly:

STEP 1: Open Android Studio
---------------------------
- Launch Android Studio from Start Menu

STEP 2: Open Your Project
-------------------------
- Click "Open an existing project"
- Navigate to: C:\Users\Dell\EZY_Booking
- Click "OK"
- Wait for Gradle sync to complete (2-3 minutes)

STEP 3: Build Release APK
-------------------------
- Go to menu: Build → Generate Signed Bundle/APK
- Choose: APK (not Bundle)
- Choose: Release (not Debug)
- It will use the keystore I created automatically
- Click "Build"

STEP 4: Get Your APK
--------------------
- APK will be created at: app\build\outputs\apk\release\app-release.apk
- Copy this file to your Android phone
- Install it
